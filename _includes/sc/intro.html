This hands-on workshop will cover basic concepts and tools around version control using Git. Participants will be encouraged to help one another and should end up walking away understanding the benefits of version control systems, and posessing competency with using Git and Github for version control. Much of the materials, including this website, is adapted from that of The Carpentries. Software Carpentry aims to help researchers get their work done in less time and with less pain by teaching them basic research computing skills. This course is a test run in preparation for my certification with The Carpentries, and paritipation in CU's program through CUIT and CU Libraries.

For more information on what The Carpentries teaches and why, please see their paper "Best Practices for Scientific Computing" and https://software-carpentry.org/about/.
